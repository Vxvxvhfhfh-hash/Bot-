# 📸 Dossier Médias

Ce dossier contient les images et médias utilisés par le bot pour créer une expérience immersive.

## 📁 Structure

```
media/
├── cyberpunk/    # Images d'ambiance cyberpunk
├── action/       # Images d'action et de combat
├── ambient/      # Images d'atmosphère générale
└── character/    # Avatars et personnages
```

## 🎨 Formats supportés

- **Images** : `.jpg`, `.jpeg`, `.png`, `.gif`, `.webp`
- **Vidéos** : `.mp4` (courtes, < 16MB pour WhatsApp)

## 🚀 Utilisation

1. **Ajouter des médias** : Placez vos fichiers dans les dossiers appropriés
2. **Détection automatique** : Le bot analyse les mots-clés pour choisir la catégorie
3. **Envoi intelligent** : 30% de chance d'envoi après chaque message RP

## 🔍 Détection des catégories

### Cyberpunk 🌃
Mots-clés : ville, néon, futur, technologie, cyber
Utilisé pour : Descriptions urbaines, technologie, ambiance futuriste

### Action ⚡
Mots-clés : combat, course, explosion, bagarre, action
Utilisé pour : Scènes de combat, poursuites, action intense

### Ambient 🌙
Mots-clés : pluie, nuit, atmosphère, ambiance, calme
Utilisé pour : Moments calmes, descriptions d'environnement

### Character 👤
Mots-clés : personnage, avatar, profil, identité
Utilisé pour : Présentations de personnages, portraits

## 💡 Conseils

- **Taille optimale** : 1920x1080 ou moins pour des temps de chargement rapides
- **Taille fichier** : < 16MB pour WhatsApp
- **Noms descriptifs** : Utilisez des noms clairs (ex: `neon_street.jpg`)
- **Qualité** : Privilégiez la qualité pour une meilleure immersion
